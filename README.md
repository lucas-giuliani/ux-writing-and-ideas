# UX, escritura e ideas

Este repositorio reúne artículos, apuntes y reflexiones sobre UX, Product Design, negocio, percepción y diseño de experiencias.

No busca reemplazar mi Medium. La idea es organizar acá una biblioteca de temas, con una breve introducción a cada texto y enlaces a las publicaciones completas.

Muchas de estas ideas nacen de situaciones cotidianas, proyectos reales o preguntas que aparecen mientras diseño. Me interesa especialmente mirar aquello que suele darse por sentado: decisiones pequeñas, fricciones invisibles y experiencias que terminan influyendo mucho más de lo que parece.

## Sobre mi manera de escribir

No escribo para convertir cada tema en una clase ni para cerrar discusiones.

Uso la escritura para ordenar ideas, revisar decisiones y compartir una mirada sobre diseño, producto y experiencias. Muchas veces el punto de partida es algo aparentemente menor: una interfaz, una conversación, un color, una espera, una promesa de marca o una situación que genera más fricción de la que debería.
